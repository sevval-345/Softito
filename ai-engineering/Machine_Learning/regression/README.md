# 📖 Hakkında

Regression klasörü, sürekli değer tahmini (Regression) problemlerine yönelik geliştirilen notebooklardan oluşmaktadır.

İçerikte doğrusal ve doğrusal olmayan regresyon modelleri, model performans analizleri ve farklı regresyon yaklaşımlarının karşılaştırıldığı uygulamalar yer almaktadır.
