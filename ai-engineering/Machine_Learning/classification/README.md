# 📖 Hakkında

Classification klasörü; klasik makine öğrenmesi algoritmalarından derin öğrenme tabanlı görüntü sınıflandırma modellerine kadar farklı yaklaşımları kapsayan notebooklardan oluşmaktadır.

İçerikte hem **Scikit-Learn** ile geliştirilen geleneksel sınıflandırma modelleri hem de **TensorFlow/Keras** kullanılarak oluşturulan derin öğrenme uygulamaları yer almaktadır. Ayrıca bilgisayarlı görü ve doğal dil işleme alanlarına yönelik örnek projeler de bulunmaktadır.

# 📚 İçerik

Bu klasörde aşağıdaki konu başlıklarına yönelik çalışmalar bulunmaktadır.

- 📌 Klasik Makine Öğrenmesi Algoritmaları
- 📌 Derin Öğrenme Temelleri
- 📌 Yapay Sinir Ağları (ANN)
- 📌 Evrişimsel Sinir Ağları (CNN)
- 📌 Transfer Learning
- 📌 Ensemble Learning
- 📌 Bilgisayarlı Görü (Computer Vision)
- 📌 Görüntü Sınıflandırma
- 📌 Yüz Tespiti (Face Detection)
- 📌 Doğal Dil İşleme (NLP)
- 📌 Duygu Analizi (Sentiment Analysis)
